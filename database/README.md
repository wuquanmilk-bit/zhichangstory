# 数据库初始化说明

## 使用方法

1. 登录 Supabase Dashboard
2. 进入左侧菜单的 "SQL Editor"
3. 点击 "New query"
4. 复制 `init.sql` 文件的内容
5. 粘贴到 SQL 编辑器中
6. 点击 "Run" 按钮执行

## 创建的表

1. **profiles** - 用户资料表
   - 与 auth.users 表关联
   - 包含用户名、头像、简介等信息
   - 用户注册时自动创建

2. **questions** - 问题表
   - 用户可以提问
   - 包含标签、浏览量、投票数等信息

## 存储配置

- 创建了 `avatars` 存储桶用于用户头像
- 配置了公开读取和用户上传权限

## 安全策略

- 使用行级安全策略 (RLS)
- 用户只能修改自己的资料
- 问题公开可见，但只有作者可以修改
